# USA-Property-Locator-Web-App
A lightweight Flask-based web application that takes a USA address as input and returns accurate latitude, longitude, and approximate property boundary dimensions using live geocoding data. The app focuses on core functionality, real-time validation, and graceful handling of invalid addresses
